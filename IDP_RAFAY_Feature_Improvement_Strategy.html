Sistem IDP Rafay sudah memiliki foundation yang solid dengan IndobERT + LayoutLMv3 untuk data extraction. Namun, gap saat ini adalah:

🔴 Current Gap

B. Confidence Score Algorithm (Pseudocode)

function calculateConfidence(field, extracted_value, original_text): base_score = IndobERT_score # 0-100% from model # Validation rules if field in required_fields and empty(extracted_value): return 0 # Total failure if not validate_format(field, extracted_value): base_score -= 25 # Template matching if template_match(field, extracted_value): base_score += 10 # Confidence adjustment if exact_match_in_original_text(extracted_value, original_text): base_score += 5 return max(0, min(100, base_score))

C. Database Schema for Corrections Log

CREATE TABLE corrections_log ( id INTEGER PRIMARY KEY, batch_id INTEGER, row_id INTEGER, field_name TEXT, original_value TEXT, corrected_value TEXT, admin_id INTEGER, timestamp DATETIME, notes TEXT, confidence_before FLOAT, confidence_after FLOAT ); -- Index untuk fast lookup CREATE INDEX idx_batch_field ON corrections_log(batch_id, field_name);
