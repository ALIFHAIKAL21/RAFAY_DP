# PROMPT GEMINI - Generate Problem Identification Sentence

Gunakan prompt ini ke Gemini untuk generate ONE sentence yang perfect untuk BAB 1.3 Identifikasi Masalah:

---

## PROMPT (Copy-Paste ke Gemini):

```
TASK: Generate ONE professional sentence yang menggambarkan identifikasi masalah operasional 
PT. Rafay Logistik untuk BAB 1.3 skripsi, dalam Bahasa Indonesia formal.

REQUIREMENTS:
- Exactly 1 sentence (tidak lebih, tidak kurang)
- Maximum 25 kata
- Bahasa Indonesia: Professional, academic, mudah dipahami
- NO predictions/proyeksi (to-the-point factual)
- Describe problem COMPLETELY dalam satu kalimat

PROBLEM DETAILS:
1. Input: WhatsApp chat unstructured text
2. Current: Pesanan 5 unit = 50 field entries (10 fields × 5 rows)
3. Error: 5-8% error rate pada field kritis
4. Complexity: Multi-slot orders + partial data + ambiguous revisions
5. Process: Manual decode → standardisasi → input Excel → validate

WHAT SHOULD THE SENTENCE COVER:
- WHAT: Unstructured WhatsApp text → field entries
- HOW MANY: 50+ entries per pesanan
- PROBLEM: Error rate 5-8% + multi-slot complexity + matching ambiguity
- SCOPE: Full picture dalam satu kalimat

STYLE:
- Academic but accessible
- Concrete numbers (50 entries, 5-8% error)
- No jargon overload
- Direct to problem root

EXAMPLE OUTPUT FORMAT:
"[Masalah X]: [Unstructured text] menghasilkan [50+ entries] dengan [error 5-8%], 
[multi-slot complexity], dan [ambiguity issue]."

NOW: Generate the FINAL ONE-LINER sentence (exactly as will appear in thesis BAB 1.3).
Output ONLY the sentence, no explanation.
```

---

## EXPECTED RESULT:

Gemini akan generate sesuatu seperti:

**"Pesanan multi-unit (5 unit = 50 entries) dari WhatsApp unstructured text menimbulkan workload manual, error 5-8%, dan ambiguitas matching antar slot/order."**

Atau variasinya yang bagus dan spesifik.

---

## LANGKAH PAKAI:

1. Copy seluruh prompt di atas (dari PROMPT hingga akhir)
2. Paste ke Gemini (gemini.google.com)
3. Tunggu output (1-2 menit)
4. Copy hasil Gemini
5. Paste langsung ke BAB 1.3 Identifikasi Masalah skripsi Anda
6. Done! ✅

---

**Pro Tip:** Jika hasil Gemini kurang sempurna, cukup kasih feedback:
- "Lebih konkrit dengan numbers"
- "Tambah aspek ambiguity matching"
- "Simplify language"
- dan Gemini akan re-generate
